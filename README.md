## Project Overview

Alien Shooter is a 2D arcade-style shooting game developed using **Python** and the **Pygame** library. The player controls a spaceship positioned at the bottom of the screen and must shoot incoming alien enemies before they reach the player. The game focuses on simple controls, real-time interaction, and classic arcade gameplay.

## Features

* Player-controlled spaceship movement using keyboard keys
* Bullet firing mechanism to destroy enemies
* Multiple alien enemies with movement patterns
* Collision detection between bullets and enemies
* Real-time score tracking
* Sound effects and background music
* Game over condition when enemies reach the player
